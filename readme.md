# InformaticaUtilGo

This is a library for Informatica in Go.

**This is a work in progress**

## Expression

Install using:
```
go get -u "github.com/michaelknowles/informaticautilgo"
```

Import using:
```
import infa "github.com/michaelknowles/informaticautilgo/expression"
```
